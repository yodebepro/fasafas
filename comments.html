Sticky Notes (Color Themes)

Digital sticky notes in six color themes for brainstorming, retros, and planning — drag to arrange, vote on favorites, and let AI cluster related notes into labeled groups automatically.

Sticky Notes
  • Create: press N (or double-click canvas) for a sticky note instantly
  • Colors: Yellow (default), Green, Blue, Pink, Orange, Purple — one keystroke per color after N
  • Type your text — auto-wraps and auto-expands
  • Font size: auto-scales to fit text, or pin at a fixed size
  • Minimum size: 100×100px. Maximum: 500×500px.
  • Move: drag by the header bar
  • Stack: drop on top of another note → creates a pile (random fan arrangement)
  • Rotate: hold Alt + drag corner handle to rotate (15° snapping by default)
Grouping & Voting
Groups
  • Select multiple notes → Group (Cmd+G) → moves as one unit
  • Named groups: click group label → rename
  • Drag notes into or out of a group
  • Nested groups: groups within groups for hierarchical organization
  • Group fill: add a background color to the group boundary
Dot Voting
  • Enable voting mode (host only): toolbar → Voting → Start Voting
  • Each participant gets 3 dots (configurable: 1–10 dots per person)
  • Click a sticky note to place a dot on it
  • Voting ends: host clicks End Voting → dots counted, notes sorted by votes
  • Export vote results to a summary Doc
AI Clustering
  • After a brainstorm: select all sticky notes → AI Cluster
  • AI uses semantic embeddings to group notes by topic similarity
  • Each cluster automatically gets a suggested label (e.g., 'Performance Issues', 'User Experience')
  • Visual: notes in each cluster arrange into a group with a colored background
  • Adjust: drag notes between clusters if AI made a mistake
  • Export cluster structure to a Doc for meeting notes
Available on Business+
AI Clustering requires AI Companion add-on or Business plan.
