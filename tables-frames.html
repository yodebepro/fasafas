Export: PNG, PDF, SVG

Export your whiteboard as a high-resolution PNG for sharing, a scalable SVG for design tools, a PDF for printing, or just send a share link for interactive browser viewing.

Export Formats
💾 Available Export Types
PNGRaster image — 1× 2× or 4× resolution. Best for: sharing in email, Slack, docs.
SVGVector — infinitely scalable, no quality loss. Best for: importing into Figma, Illustrator, design tools.
PDFMulti-page PDF — one frame per page (if frames are used), or full canvas as one page.
.whiteboardFasaFas native format — can be imported back or shared with collaborators.
OPMLFor mindmaps — importable into XMind, Miro, Mindmeister.
Export Options
  • Export scope: Entire canvas · Current viewport only · Selected objects only
  • Resolution (PNG): 1× (screen), 2× (retina/print), 4× (large format print)
  • Background: include background color, transparent, or white
  • Grid lines: include or exclude canvas grid in export
  • Watermark: remove FasaFas watermark on Pro+ plans
  • Batch export: if frames are used → export as a ZIP of individual frame PNGs
  • Auto-export: configure automatic export to Google Drive or OneDrive on a schedule
Share Link (Interactive View)
  • Generate a shareable link: toolbar → Share → Copy Link
  • Recipients open the link in a browser — no FasaFas account required for view-only
  • Interactive: they can pan and zoom the canvas freely
  • Named version link: share a link to a specific historical snapshot
  • Password protection: optional — set a password for the share link
  • Expiry: set link to expire after 7 days, 30 days, or never
  • Embed: paste the embed code to show an interactive whiteboard in a website or Notion page
