Whiteboard per Room

Every breakout room has its own dedicated whiteboard — a full infinite collaborative canvas where participants can brainstorm, diagram, and visualize ideas together in real time.

Enabling Whiteboard in Breakout Rooms
  1. Before rooms open: configure settings
    In the Breakout Rooms panel, click Options → enable Allow whiteboards in breakout rooms. This allows participants to open the whiteboard inside their room.
  2. Participants open their room whiteboard
    Inside the room, click Whiteboard in the meeting toolbar (or More → Whiteboard). A fresh, blank whiteboard opens specific to that room.
  3. All room participants collaborate
    All participants in the same room see and edit the same whiteboard in real time. Participants in other rooms have their own separate whiteboard.
Host can see all room whiteboards
By joining any breakout room, the host can view and participate in that room's whiteboard. If you want to review all room whiteboards after the session, they are saved to Hub → Whiteboards, labeled by room name.
Whiteboard Tools Available in Breakout Rooms
Drawing & Annotation
  • Pen / freehand drawing
  • Highlighter (multiple colors)
  • Shapes: rectangles, circles, arrows, lines
  • Smart connectors (auto-route between shapes)
  • Text tool (bold, italic, font size)
  • Sticky notes (color-coded)
  • Eraser (object or freehand)
Collaboration Features
  • Named cursors: see who is drawing what
  • Real-time sync (CRDT — no conflicts)
  • Undo / Redo (per user)
  • Zoom in/out and pan (infinite canvas)
  • Lock objects (prevent accidental edits)
  • Laser pointer (presentation mode)
  • Templates: sticky note grid, Kanban, mindmap
Saving and Sharing Room Whiteboards
  • Room whiteboards save automatically throughout the session
  • When rooms close, all whiteboards are saved to Hub → Whiteboards
  • Named by room: "Breakout Room 1 — April 30", "Group A — Sprint Review", etc.
  • Host can open any room's whiteboard after the meeting ends
  • Host can share individual room whiteboards with any participant via link
  • Export: PNG, PDF, or SVG from the whiteboard menu
  • Whiteboard can be embedded in a FasaFas Doc for post-meeting reference
  • Whiteboard content NOT included in the meeting recording automatically — screenshot or export manually
