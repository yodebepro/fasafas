Return to Main

When your breakout session is complete, the host can return every participant from every room back to the main session with a single click — instantly and simultaneously.

Closing All Rooms and Returning Everyone
How to Close All Rooms
  1. From the main session (recommended)
    Open the Breakout Rooms panel → click Close All Rooms. A countdown begins (60 seconds by default) and all participants are returned when it hits zero.
  2. From inside a breakout room
    The host can also click Close All Rooms from inside a room. They return to the main session along with everyone else.
  3. Countdown appears
    Every participant sees "Breakout rooms will close in 60 seconds. Please wrap up." banner. The countdown is customizable (10s, 30s, 60s, 2 minutes).
  4. All participants return
    At 0:00, all rooms close. Every participant automatically reconnects to the main session's audio and video. Room audio and video disconnects from the breakout and reconnects to main — usually under 3 seconds.
Immediate Close (No Countdown)
  • Enable Immediate Close in Breakout Room options before opening rooms
  • Participants have 0 seconds — moved instantly with no warning
  • Use for time-critical situations or when a strict schedule must be met
  • Not recommended for normal sessions — abrupt exits can be disorienting
Countdown Behavior Explained

Participant displayOrange banner at top of screen with live countdown timer
Cancel optionHost can cancel the countdown and keep rooms open (within the countdown period)
Audio cueChime plays when countdown starts + when rooms close
Can the host cancel a close?
Yes. If you click Close All Rooms accidentally or need more time, click Cancel Close in the Breakout Rooms panel before the countdown ends. Rooms stay open and participants see the countdown banner disappear.
After Everyone Returns to Main
Immediate Post-Return
  • All participants are back in the main session's audio and video
  • Host's video appears full-screen (if they were the active speaker)
  • In-meeting chat from the main session is visible again
  • Breakout room chats are separate — not merged into main chat
  • The main session timer resumes from where it was
  • Host can start debriefing, share-outs, or the next agenda item immediately
Room Data Preserved After Close
  • Room whiteboards saved to Hub automatically
  • Room chat histories accessible via Breakout Rooms history (for host)
  • Any files shared to rooms remain in participants' Hub
  • AI summaries of breakout room discussions (if AI enabled per room)
  • Participant time spent in each room logged in the attendance report
  • Rooms can be reopened within the same meeting session if needed
