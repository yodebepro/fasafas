Create Rooms

FasaFas lets you instantly split any meeting into up to 50 separate rooms, each with its own video, audio, chat, and whiteboard. Rooms can be created before the meeting or on the fly during a live session.

How to Create Breakout Rooms
During a Live Meeting
  1. Open the Breakout Rooms panel
    Click the Breakout Rooms button in the bottom toolbar. If it's hidden, click More (···) → Breakout Rooms. Only the host or co-host can access this.
  2. Set the number of rooms
    Type a number or use the +/− controls. You can create 2–50 rooms. FasaFas shows a preview of how many participants will be in each room.
  3. Choose assignment mode
    Select Automatically, Manually, or Let participants choose. This can be changed before you open rooms.
  4. Open All Rooms
    Click Open All Rooms. Each participant receives a blue invite dialog to join their assigned room. They have 60 seconds before being automatically moved.
Before the Meeting (Pre-configure)
  1. Open the Schedule Meeting form
    Go to Meetings → Schedule a Meeting. Scroll to Advanced Options and enable Breakout Rooms.
  2. Pre-assign participants
    Click Manage Breakout Rooms. Import a CSV or manually type participant emails and assign them to named rooms.
  3. Save as template
    Click Save. The room setup is saved with the meeting. For recurring meetings, the same assignments reload every time.
Can I create more rooms after they're open?
Yes. While rooms are active, click Recreate in the Breakout Rooms panel to add more rooms or adjust the count. Existing rooms and their participants are unaffected — new rooms are added alongside them.
