📥 Available Recording Output Files
MP4 (Video + Audio)Full meeting recording with selected layout. H.264 encoded. ~0.8 GB/hour at 1080p.
M4A (Audio Only)AAC audio from all speakers mixed together. ~50 MB/hour. Great for podcast-style meetings.
VTT (Captions)WebVTT caption file with speaker labels and timestamps. Works in YouTube, Premiere Pro, Final Cut.
SRT (Subtitles)SubRip format — compatible with VLC, DaVinci Resolve, social media upload.
JSON (Structured)Full transcript with word-level timestamps, speaker IDs, confidence scores.
Chat (TXT)All in-meeting chat messages with sender name and timestamp. Plain text.
Summary (PDF/DOCX)AI summary formatted as a professional document with all sections and action items.
How to Download Recordings
  1. Go to Hub → Recordings or Meetings → [Past Meeting] → Recordings tab
  2. Click the ⬇ Download button on the recording tile
    A download options panel appears listing all available files for this recording.
  3. Select the file(s) you want
    Check one or multiple formats. Click "Download Selected" to start downloading all checked files.
  4. Files arrive in your Downloads folder
    Large MP4 files may take a few minutes depending on your connection speed. VTT, JSON, and TXT files download instantly.
Download restrictions
Admin can restrict MP4 download to host-only. Participants can still watch via the streaming player but cannot download the file. VTT and TXT files may still be downloadable even when MP4 is restricted, depending on admin settings.
Batch Recording Export
  • Export multiple recordings at once: Hub → Recordings → select checkboxes → "Download Selected"
  • ZIP file created containing all selected files (all formats you specify)
  • Batch export via API: GET /api/v1/recordings?date_from=&date_to= with format parameters
  • Scheduled export: admin can configure daily/weekly export to a customer-owned S3 bucket
  • Compliance export: eDiscovery API allows bulk export for legal hold purposes
  • Large exports (>5 recordings or >1 GB) are processed asynchronously — you receive an email with a download link when ready
