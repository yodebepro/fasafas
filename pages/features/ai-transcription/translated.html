Translated Captions

The Translated Captions add-on displays live captions in any of 30+ languages — so participants can follow a meeting conducted in a language they don't speak.

How Live Translation Works
  1. Whisper transcribes the original language speech
    The same Whisper pipeline that powers standard captions produces the original-language text with ~1–2s latency.
  2. Translation API called per caption segment
    Each caption segment is sent to DeepL (primary — highest quality for European languages) or Google Translate (fallback, better for Asian languages). Best-of routing selects the provider per language pair.
  3. Translated text returned in ~200–400ms
    Translation adds 200–400ms to the total caption delay. Total display latency: 1.5–2.5 seconds behind actual speech.
  4. Each participant selects their preferred caption language
    Translation is applied per-subscriber independently. Participant A sees French captions while Participant B sees Spanish — from the same English-speaking presenter.
Supported Languages
High Quality (DeepL)
  • English, French, German, Spanish, Portuguese
  • Italian, Dutch, Polish, Russian, Swedish
  • Norwegian, Danish, Finnish, Romanian, Hungarian
  • Czech, Slovak, Bulgarian, Greek, Turkish
  • Chinese (Simplified), Japanese, Korean
Good Quality (Google Translate)
  • Arabic, Hebrew, Hindi, Bengali, Urdu
  • Indonesian, Malay, Vietnamese, Thai
  • Swahili, Amharic, Yoruba, Hausa
  • Ukrainian, Croatian, Serbian, Slovenian
  • Any language pair not covered by DeepL
