Dynamic Gallery (AI)

Dynamic Gallery is FasaFas's AI-powered layout that automatically sizes video tiles based on who's most active — giving more screen space to recent speakers and active participants.

How Dynamic Gallery Works
  1. AI scores each participant continuously
    Every 2 seconds, the system calculates a "visual prominence score" for each participant: speech activity (60% weight) + recent interaction (reactions, hand raise: 20%) + host setting (20%).
  2. Tile sizes assigned by score
    The highest-scoring participant gets the largest tile (up to 2× normal size). Lower-scoring participants get smaller tiles. Everyone is still visible.
  3. Layout recalculates on score change
    When scores change significantly, tiles smoothly resize and reposition with a 0.5s CSS animation — no jarring jumps.
  4. Minimizes empty black space
    Participants with video off are shown as smaller name tiles, freeing space for active camera users. No large black squares wasting screen real estate.

Enabling Dynamic Gallery
  • Click View in the meeting toolbar → select "Dynamic Gallery"
  • Or in Settings → Video → Layout → set Default Layout to "Dynamic Gallery"
  • Admins can set Dynamic Gallery as the account-wide default for all meetings
  • Each participant can override to their preferred layout locally — their choice doesn't affect others
  • Available on Pro, Business, and Enterprise plans
  • Compatible with desktop app and web browser (Chrome/Edge/Firefox/Safari)
