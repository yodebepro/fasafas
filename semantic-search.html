File Sharing

Share files in any channel or DM with a simple drag-and-drop — with inline previews, version tracking, and direct links to Google Drive or OneDrive files without downloading.

File Management
File Sharing Features
  • Drag files directly into any chat window to upload
  • Or click the paperclip icon → browse files
  • Images and videos show inline previews — no download needed
  • PDFs render inline with page navigation
  • Maximum file size: 500 MB per file (1 GB for video on Business+)
  • Files searchable by name across all channels
  • Annotate images: click the pen icon on an image → draw, highlight, add text
  • Comment on specific sections of a document
Supported File Types
With Inline Preview
  • Images: JPG, PNG, GIF, WebP, SVG
  • Videos: MP4, MOV, WebM (plays inline)
  • PDFs: page-by-page navigation
  • Office docs: DOCX, XLSX, PPTX (read-only preview)
  • Code files: syntax-highlighted preview
  • Audio: MP3, M4A, WAV (plays inline)
Cloud File Links
  • Google Docs / Sheets / Slides — previews without downloading
  • Microsoft Word / Excel / PowerPoint via OneDrive — rich unfurl
  • Notion pages — title + description unfurl
  • GitHub files and PRs — status + diff unfurl
  • Figma frames — live thumbnail preview
File Management
  • File library: sidebar → Files → search, filter by type, uploader, date
  • Version history: re-upload the same filename → old versions preserved and linked
  • Download: hover any file → Download button
  • Delete: only the uploader or admin can delete a file
  • File retention: admin sets how long files are stored (30 days → forever)
  • Storage quota: counted against cloud storage plan (same pool as recordings)
