Writing Mode

Open any FasaFas Doc and activate Writing Mode — Claude appears as an intelligent co-author that suggests next sentences, rewrites selections, adjusts tone, and improves clarity, all inline without leaving the document.

Technical Details
Writing Mode Features
Inline Suggestions
  • AI suggests the next sentence as you type (ghost text)
  • Press Tab to accept, Esc to dismiss, keep typing to ignore
  • Suggestions adapt to your writing style as the session continues
  • Context-aware: reads surrounding paragraphs for consistency
  • Works in headings, body, bullet lists, and table cells
Selection-Based Actions
  • Highlight any text → floating AI toolbar appears
  • Actions: Rewrite, Shorten, Expand, Fix Grammar, Change Tone, Translate
  • Each action shows a preview before applying
  • Accept → replaces selection. Retry → generates different version
  • Full undo/redo — AI changes part of document history
Slash Commands
💡 Available in Writing Mode
/ai [prompt]Generate text at cursor — anything you describe
/rewriteRewrite current paragraph for clarity and flow
/shortenCondense selected text ~50% without losing meaning
/expandAdd detail and depth to a brief paragraph
/tone formalRewrite to professional/formal register
/tone casualRewrite to conversational, friendly tone
/translate [lang]Translate selection into specified language
/continueGenerate next paragraph continuing current thought
/research [topic]Trigger Deep Research and insert into doc
/summaryGenerate TL;DR of the entire document
Technical Details
  • Model: Claude 3.5 Sonnet — best at following stylistic instructions
  • Streaming SSE: text appears word-by-word for a natural co-writing feel
  • Context window: up to 8,000 tokens of surrounding doc content per request
  • Offline-safe: unsaved AI suggestions persist as ghost text until confirmed
  • Admin can restrict Writing Mode per group: Admin → AI Settings → Writing Mode
  • Not available in read-only or comment-only permission modes
