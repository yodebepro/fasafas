Create Events + Auto Meeting Link

Create a calendar event and FasaFas automatically generates and attaches a meeting link — guests receive the invite via email with a one-click join link, no separate scheduling step needed.

Creating a Calendar Event
  1. Click any time slot in the calendar or press N for New Event
  2. Fill in: Title, Date & Time, Duration, Location/Video
  3. Add invitees: type name or email — suggests from your contacts and company directory
  4. Set RSVP expectation: Required / Optional / Informational
  5. Add description, agenda items, and pre-read links
  6. Click Save → event created + invitations sent automatically
Auto Meeting Link
  • FasaFas auto-generates a meeting room link for every event with 2+ invitees
  • Link format: fasafas.app/j/[unique-room-id]
  • Link and meeting ID are included in the calendar invite email automatically
  • One-click join: from calendar → click 'Join Meeting' button on event detail
  • Meeting room persists for the duration of the event + 30 minutes after
  • Admin can configure: always add meeting link, only when invited, or never (manual)
Replacing Google Meet or Zoom Links
If you use Google Calendar sync, FasaFas can replace the Google Meet link in synced events with a FasaFas link — configure in Settings → Calendar → Meeting Link Override.
