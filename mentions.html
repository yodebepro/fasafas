Direct Message Features
  • Start a DM: click + New Message → search for a teammate
  • Full markdown support: **bold**, *italic*, `code`, links, lists
  • Rich media previews: links, images, PDFs show inline previews
  • Reactions: hover any message → emoji picker → react
  • Edit or delete your own messages (edit history shown to recipient)
  • Star important messages → accessible in 'Starred Items'
  • Pin messages within a DM for quick reference
  • Search DM history by keyword or date range
Group DMs (Up to 9 People)
  • Create a group DM: New Message → add multiple names
  • Up to 9 participants in a group DM
  • Convert a group DM to a private channel if the conversation grows
  • Participants list shown in the header — click to view profiles
  • '@here' notifies all group DM participants
  • Add or remove participants (adding shows conversation from that point; history not shared retroactively)
  • All file sharing, reactions, and threading features available in group DMs
Escalate DM to Video Meeting
  • Phone/video icon in the DM header → starts an instant FasaFas meeting with that person
  • All participants in the DM are invited automatically
  • Meeting runs in the full FasaFas meeting room — camera, mic, screen share, recording
  • Or: share a scheduled meeting link directly in the DM
  • After the meeting: AI summary automatically posted back into the DM thread
