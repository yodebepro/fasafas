Raise Hand

Participants in any breakout room can raise their hand to signal they need help. The host receives an instant notification from the main session and can jump to that room immediately.

Help Flow
How Participants Raise Their Hand
Inside a Breakout Room
  • Click the Ask for Help button at the top of the room screen
  • The button sends a help request to the host in the main session
  • A confirmation message appears: "Your help request was sent to the host"
  • The room continues normally — no disruption to other participants
  • If the host joins within 60 seconds, the request clears automatically
  • Participants can send multiple requests if the host hasn't responded
  • Individual hand raise: participants can also raise their personal hand (for speaking) — separate from the room help request
Personal Raise Hand vs. Room Help Request
  • Room Help Request ("Ask for Help") — signals the entire group needs the host. Host jumps to the room.
  • Personal Raise Hand — signals you want to speak within your current room. Other room participants see your hand icon.
  • Both can be active simultaneously
  • Both are visible in the Participants panel within the room
What the Host Sees
Notification in Main Session
  • A notification banner appears: "Group A is asking for help"
  • The Breakout Rooms panel shows a 🙋 icon next to the room name with a request
  • An audio chime plays (distinct from other notifications)
  • The notification stays visible until the host joins that room or dismisses it
  • Multiple rooms can have active help requests simultaneously — all listed
  • Requests are listed in time order (first request first)
🔔 Notification Details
LocationTop of Breakout Rooms panel + banner notification
SoundDistinct chime (different from meeting chat notification)
PersistenceStays active until host joins or 10 min timeout
ActionClick "Join" in the notification to go directly to that room
Complete Help Request Flow
  1. Participant clicks "Ask for Help"
    A help signal is sent instantly from the breakout room to the main session. The participant sees "Help requested — the host will join soon."
  2. Host receives notification
    A chime plays in the main session and a banner appears: "Group A is asking for help." The Breakout Rooms panel highlights the requesting room with a 🙋 badge.
  3. Host clicks Join in the notification
    One click takes the host directly into the requesting room. Faster than scrolling the Breakout Rooms panel.
  4. Host assists the group
    Host speaks, shares screen, writes on whiteboard, or guides the discussion. The help request clears from the panel.
  5. Host leaves when done
    Clicks "Leave Breakout Room" to return to main session and attend to other rooms or the next help request.
