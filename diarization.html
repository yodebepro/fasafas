Live Captions

Live captions appear as a scrolling text overlay at the bottom of the screen — updated every few seconds as speech is recognized. Every participant can toggle captions independently.

Caption Display Options
Caption Overlay
  • Displayed at the bottom of the video area as a semi-transparent text bar
  • Shows the last 2–3 sentences of recognized speech
  • Updates approximately every 0.5–1 second as speech is processed
  • Total display latency: 1–2 seconds behind actual speech
  • Speaker name shown before each caption line (if diarization is enabled)
  • Your own speech is highlighted in a different color
Caption Customization
  • Font size: Small / Medium / Large / Extra Large
  • Background opacity: transparent → dark semi-transparent → solid
  • Position: bottom (default) / top / floating overlay
  • Caption history: open the full transcript sidebar to see all past lines
  • Show / hide speaker labels in caption bar
  • Captions persist on screen for 3 seconds after each update then fade
Turning Captions On and Off
  • Toggle captions: click More (···) in the meeting toolbar → Show Captions
  • Each participant independently decides whether to see captions — it's a local setting only
  • Host can enable captions for all participants via meeting settings (forces it for everyone)
  • Admin can set "Enable live transcription for all meetings" account-wide (admin panel → AI Settings)
  • Consent required: a banner shows "This meeting is being transcribed" when AI transcript starts — required for all plans
  • Turning off captions doesn't stop the transcript — it just hides the overlay for you
Accessibility Benefits
  • WCAG 2.1 AA compliant caption rendering
  • High-contrast mode: white text on black background option for low-vision users
  • Font family can be set to a dyslexia-friendly font (OpenDyslexic) in accessibility settings
  • Screen reader integration: caption text exposed via ARIA live region for screen reader users
  • Caption history accessible via keyboard shortcut (Alt+C on Windows)
  • Translated Captions add-on allows deaf users to read in their preferred language
