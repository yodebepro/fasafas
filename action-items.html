Grammar & Style Suggestions

Real-time grammar and style suggestions appear as underlines as you type — accept a fix with one click, build a custom dictionary for your company's terms, and check readability scores for any selection.

Grammar & Spelling Suggestions
  • Real-time: suggestions appear as colored underlines as you type (no manual scan needed)
  • Red underline: spelling error
  • Yellow underline: grammar issue (e.g., subject-verb agreement, tense inconsistency)
  • Blue underline: style suggestion (e.g., passive voice, wordiness)
  • Hover any underline → suggestion popup with fix + explanation
  • Click 'Accept' → fix applied. 'Dismiss' → suggestion suppressed for this word/phrase.
  • 'Add to dictionary' → word added to your custom vocabulary — no longer flagged
Style Guide Enforcement
  • Admin can upload a custom style guide: Admin Panel → Docs → Style Guide
  • Rules examples: 'Use Oxford comma', 'Avoid passive voice', 'Always capitalize Product names', 'Use US English spelling'
  • Violations highlighted with a purple underline — distinct from grammar (yellow) and spelling (red)
  • Style suggestions are suggestions only — cannot be set as hard blocks
  • Team-wide: style guide applies to all workspace members' docs automatically
  • Import: style guide can be a plain text or JSON rule set — contact support for the format spec
Readability Score
  • Select any text → right-click → Readability Score (or Doc menu → Analyze → Readability)
  • Shows: Flesch Reading Ease (0–100), Flesch-Kincaid Grade Level, and Gunning Fog Index
  • Interpretation: FRE 60–70 = plain English · below 30 = academic/complex
  • Suggestions: 'This selection has an average sentence length of 28 words — consider shorter sentences'
  • Grade level target: set a target grade level in Doc settings → suggestions guide you toward that level
  • Overall doc score: shown in Doc Analytics panel — tracks score over time as you edit
