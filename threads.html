Action Item Extraction from Threads

AI scans any channel thread or conversation for action items — commitments, assignments, and deadlines — and surfaces them as structured tasks with one click to push to the Tasks module.

Action Item Extraction from Chat
  • Right-click any thread → 'Extract action items with AI'
  • Or hover a message → ⋯ → 'Find action items'
  • AI scans the thread from beginning to end — not just the hovered message
  • Results shown in a sidebar panel: each action item with task, assignee, due date
  • Confirm which ones to keep → push to Tasks or dismiss
  • Channel-level: weekly digest of all action items extracted from all channel threads (optional, admin toggle)
What Gets Detected
  • Commitments: 'I'll update the doc by EOD', 'will send you the contract today'
  • Assignments: 'Can you review the PR?', '@Ana please handle the client call'
  • Deadlines: 'needs to be done by Friday', 'before the board meeting on the 15th'
  • Implicit tasks: 'we need someone to write the announcement' — AI flags as unassigned
  • Context preserved: each item includes the original message quote and a link to that message in the thread
Confidence Scores
Each detected item has a confidence rating (High / Medium / Review-needed). Low-confidence items are shown separately for your review before being saved.
Push to Tasks
  • One-click 'Push to Tasks' sends the item to the FasaFas Tasks module
  • Assignee pre-filled (if detected), due date pre-filled (if mentioned), source thread linked
  • Assignee receives a notification: 'Ana assigned you a task from #product channel: Review the PR'
  • Integrations: push to Asana, Jira, Monday.com, Notion, or Zapier webhook instead
  • Weekly channel digest: admin can enable a Monday 9 AM post to each channel listing all open items from that channel
