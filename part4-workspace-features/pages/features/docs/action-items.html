Action Items → Tasks

AI scans any doc for action items — explicit tasks, commitments, and follow-ups — and surfaces them as a checklist you can push to the Tasks module with assignee and due date pre-filled.

Action Item Extraction in Docs
  • Trigger: Doc menu → Extract Action Items (runs AI scan on entire doc)
  • Or: select a section → right-click → Extract Action Items from Selection
  • AI looks for: commitment language, assignment phrases, deadline mentions
  • Results shown in a right panel: each item with task, assignee (if mentioned), due date (if mentioned), and the source paragraph
  • Confidence badge per item: High / Medium / Review-needed
  • Run after every edit: 'Re-scan' button refreshes the list if you've added content
Checklist Interface
  • Extracted items shown as an interactive checklist in the right panel
  • Check off items as you complete them — status syncs to anyone viewing the doc
  • Assign: click the assignee field → search for a workspace member
  • Due date: click date field → date picker
  • Edit: click the task text to modify it
  • Delete: click × to remove a false positive from the list
  • Checklist also insertable inline: 'Insert as checklist block' → adds the full list into the doc body
Push to Tasks & Integrations
  • Push All: one click → all items sent to FasaFas Tasks module with assignee + due date + doc link
  • Push selected: check specific items → 'Push Selected'
  • Assigned person notified: push notification + email with task details and link back to doc
  • Integrations: push to Asana, Jira, Monday.com, Notion, or Linear via integrations panel
  • Webhook: POST action items to your endpoint on extraction (for custom CRM or project management systems)
  • Recurring doc: set auto-extraction + auto-push for docs like weekly status updates
