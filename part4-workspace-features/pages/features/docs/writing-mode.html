Writing Mode: AI Co-Author

Activate Writing Mode in any doc and Claude becomes your co-author — suggesting next sentences as ghost text, rewriting selected paragraphs, adjusting tone, and improving clarity all inline.

Under the Hood
AI Writing Mode
Ghost Text Suggestions
  • AI suggests the next sentence as you type — appears as faded ghost text after your cursor
  • Press Tab to accept the full suggestion
  • Press → to accept word by word
  • Press Esc to dismiss
  • Adapts to your writing style during the session
  • Works in all block types: paragraphs, bullets, headings, table cells
Selection-Based Actions
  • Highlight text → floating toolbar appears above selection
  • Rewrite — improve clarity and flow
  • Shorten — reduce length by ~50%
  • Expand — add depth and detail
  • Tone: Formal / Casual / Persuasive / Empathetic
  • Translate — to any supported language
  • Fix Grammar — correct errors without changing meaning
Writing Mode Slash Commands
💡 Slash Commands (Writing Mode)
/ai [prompt]Generate content at cursor from any description
/rewriteRewrite current paragraph
/shortenCondense current paragraph ~50%
/expandAdd depth to current paragraph
/tone formalFormal/professional register
/tone casualConversational, friendly tone
/translate [lang]Translate selected text
/continueGenerate next paragraph
/research [topic]Trigger Deep Research and insert results
/summaryInsert TL;DR of entire document at top
/outlineGenerate a document outline from current content
Technical Details
  • Model: Claude 3.5 Sonnet — best at stylistic instruction-following
  • Streaming: text appears word-by-word via SSE — natural co-writing feel
  • Context: up to 8,000 tokens of surrounding content sent per request
  • Undo: AI changes are part of document history — Cmd+Z reverts any AI suggestion
  • Admin control: enable/disable Writing Mode per user group
  • Privacy: doc content sent to Claude API — subject to FasaFas's data processing agreement
  • Not available in read-only or comment-only permission modes
