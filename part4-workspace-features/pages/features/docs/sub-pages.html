Nested Sub-Pages

Organize your workspace like a wiki — create nested sub-pages inside any doc, build a hierarchy as deep as you need, and navigate with the page tree sidebar and breadcrumb trail.

Creating Sub-Pages
  • In any doc: type /page → 'New sub-page' → give it a title → press Enter
  • Or: left sidebar → hover a page → + icon → 'New page inside'
  • Sub-page opens in its own editor — fully independent content
  • Breadcrumb at the top shows: Workspace → Parent Doc → Sub-Page
  • Nest as deep as needed: sub-pages can have their own sub-pages (unlimited depth)
  • Move a page: drag in sidebar to a new parent, or right-click → Move to… → pick destination
  • Duplicate: right-click any page → Duplicate → creates a copy with all sub-pages
Navigation
  • Left sidebar: collapsible page tree — click arrows to expand/collapse folders
  • Search: Cmd+K → type page title → instant navigation
  • Breadcrumb: click any crumb to jump up the hierarchy
  • Back/Forward: browser back button or Alt+← / Alt+→
  • Linked mentions: type @PageTitle in any doc → creates an inline link to that page — click to navigate
  • Backlinks: every page shows 'Referenced by' — all pages that link to this one
  • Pin: pin any page to the top of your sidebar for quick access
