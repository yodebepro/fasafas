Deep Research: Multi-Source Reports

Trigger a Deep Research report from inside any doc — AI decomposes your question, browses the web, reads sources, and inserts a structured, cited report directly into the document.

Deep Research in Docs
  • Trigger: type /research [your question] in any doc
  • Or: AI Assistant panel → Deep Research tab → enter question → 'Insert into Doc'
  • A progress panel shows research steps as they complete (30 seconds – 5 minutes depending on depth)
  • Result inserted at cursor position as a structured doc section: H2 title, summary paragraph, detailed sections, sources list
  • Result is editable: treat it as a starting point — edit, remove, or expand any section
  • Re-run: hover the inserted section → 'Update Research' → refresh with latest web data
Research Pipeline
  1. Query decomposition into sub-questions (GPT-4o)
    5–12 sub-questions identifying different research angles
  2. Parallel web search per sub-question
    Top 3–5 sources read in full per angle
  3. Source extraction and cross-referencing (Claude)
    Key claims extracted, conflicts flagged, gaps identified
  4. Structured report synthesis
    Executive summary + sections + citations + confidence scores
  5. Inserted into doc at cursor
    Formatted as proper H2/H3 hierarchy — not a wall of text
Citation Format
  • Inline citations: superscript numbers [1] [2] linked to the sources list at bottom
  • Sources section: automatically appended — title, URL, author, date, relevance score
  • Hover any citation → tooltip shows source title and URL
  • Click citation → opens source in new tab
  • Export: when doc is exported to PDF, citations formatted as footnotes or endnotes
  • Copy citations: 'Copy as APA' / 'Copy as MLA' / 'Copy as BibTeX' from the sources section
