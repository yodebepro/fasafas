Mindmap Auto-Layout

Build mindmaps with keyboard-first creation — press Tab to add a child branch, Enter to add a sibling — and let auto-layout arrange the tree structure as you think.

Building a Mindmap
⌨️ Mindmap Keyboard Shortcuts
TabAdd a child branch to the selected node
EnterAdd a sibling branch (same level) below the selected node
DeleteRemove the selected node and all its children
BackspaceRemove just the selected node — promotes children to parent level
SpaceEdit the text of the selected node
Collapse/ExpandClick the small circle on a node to collapse/expand its children
Layout Options
🧠 Mindmap Layouts
Horizontal (default)Root in center, branches spread left and right — classic mind map
VerticalRoot at top, branches grow downward — like an org chart
Right-onlyRoot on the left, branches only on the right — outline style
RadialRoot in center, branches in all directions — organic look
Auto-balanceFasaFas balances branch distribution automatically to minimize whitespace
Mindmap Export
  • Export as image: PNG, SVG — preserves visual layout
  • Export as outline: converts to a numbered/bulleted markdown outline
  • Push to Doc: creates a FasaFas Doc with headings matching the mindmap structure
  • Export to OPML: import into mind-map tools like XMind, Miro, Mural
  • Share link: anyone with the link can view the mindmap in their browser (no account required for view-only)
