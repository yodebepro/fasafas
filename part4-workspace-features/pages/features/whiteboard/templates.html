Using Templates
  1. New whiteboard → 'Browse Templates' (or press T from an existing whiteboard)
  2. Browse by category or search by keyword
  3. Preview: hover any template → see a full-size preview
  4. Insert: click 'Use Template' → template placed on canvas at your current viewport
  5. Fully editable: every element — text, color, layout — can be modified after insert
  6. Multiple templates: insert as many as you like on the same infinite canvas

Custom & Brand Templates
  • Create your own: design a whiteboard → toolbar → Save as Template
  • Private template: only visible to you
  • Shared template: visible to all workspace members
  • Admin templates: admins can push templates to all users — for standardized meeting formats
  • Template variables: embed {meeting_title}, {team_name}, {date} — auto-filled when the template is used
  • Export template: download as a .whiteboard file and share externally
  • Import template: drag a .whiteboard file onto the canvas to import
