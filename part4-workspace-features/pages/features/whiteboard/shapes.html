Shapes: Rect, Circle, Diamond, Arrow

A complete set of shapes for diagrams and flowcharts — rectangles, circles, diamonds, triangles, and arrows — each with fill, stroke, text, and customizable style.

Shape Library
Basic Shapes
  • Rectangle (rounded or sharp corners)
  • Circle / Ellipse
  • Diamond (for flowchart decisions)
  • Triangle (right / equilateral / isosceles)
  • Hexagon, Pentagon, Star (2–12 points)
  • Cross / Plus sign
  • Cloud / Callout / Speech bubble
Flowchart Shapes
  • Process (rectangle) — standard step
  • Decision (diamond)
  • Terminator (rounded ends) — start/end
  • Parallelogram — data input/output
  • Document shape
  • Database (cylinder)
  • Swimlane container (auto-resizes to content)
Shape Styling
🎨 Style Properties
FillSolid color, gradient (linear/radial), or none (transparent)
StrokeColor, width (0–20px), dash pattern (solid/dashed/dotted)
Opacity0–100% for the entire shape
ShadowDrop shadow with configurable offset, blur, and color
Corner radiusFor rectangles: 0–50px corner rounding
Color presets8 quick-access color swatches per brand theme, set by admin
Text Inside Shapes
  • Double-click any shape to enter text-editing mode
  • Text inherits shape fill for contrast (auto-detects dark/light background)
  • Font: size, weight, italic, underline — per shape
  • Text alignment: horizontal (left/center/right) and vertical (top/middle/bottom)
  • Auto-size: shape auto-expands to fit text, or set a fixed size
  • Overflow: text that exceeds fixed-size shape truncates with '...' indicator
  • Markdown in text: **bold**, *italic*, `code` rendered inline
