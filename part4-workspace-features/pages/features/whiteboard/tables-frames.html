Tables, Frames & Image Insert

Structure your canvas with tables for data, frames to organize sections, and embedded images or code blocks — making whiteboards work for technical documentation and structured planning.

Tables on the Canvas
  • Insert: toolbar → Table → choose rows × columns (up to 20×20)
  • Click any cell → type text, numbers, or paste content
  • Resize rows/columns: drag the border between cells
  • Add row: click + at the bottom of the table. Add column: click + at the right.
  • Delete row/column: right-click header → Delete Row / Column
  • Sort column: right-click column header → Sort A→Z or Z→A
  • Table text supports: **bold**, *italic*, `code` markdown
  • Style: header row can be styled differently (bold, colored background)
Frames (Sections)
  • Frames are named sections that organize areas of the canvas into labeled slides
  • Create: press F → drag to draw frame area
  • Name: double-click frame header → rename
  • Frame navigator: left sidebar lists all frames — click to jump to any frame
  • Use for: separate sections of a Kanban, individual retro topics, slide-by-slide presentation
  • Present mode: frames become slides — navigate frame-by-frame for a presentation
Images & Embeds
  • Insert image: paste from clipboard, drag-and-drop from your computer, or insert from URL
  • Supported formats: JPG, PNG, GIF (animated), WebP, SVG
  • Resize: drag corner handle — maintains aspect ratio unless Shift held
  • Embed FasaFas Recording: paste meeting recording URL → shows thumbnail player inline
  • Embed YouTube/Loom: paste URL → inline playable video card
  • Code block: insert a code block with syntax highlighting — good for technical whiteboards
