Comments on Canvas Objects

Leave threaded comments directly on any canvas object — shapes, sticky notes, or frames — with @mentions, resolution tracking, and links back to the specific object.

Adding Comments to Canvas Objects
  • Right-click any object → Add Comment
  • Or: select an object → C keyboard shortcut
  • Comment panel opens on the right side
  • Type your comment — markdown supported, @mention supported
  • Submit → comment appears as a badge on the object (orange dot)
  • Reply: click the object's comment badge → reply in the thread
  • All comments associated with the object, not a canvas position — moves with the object if dragged
Managing Comments
  • Comment panel (right sidebar) lists all open comments across the canvas
  • Filter: by open / resolved / @mentioned me
  • Click any comment → canvas pans to the associated object
  • Resolve: click ✓ on a comment thread → it's marked resolved and hidden from default view
  • Reopen: filter to show resolved → click Reopen on any comment
  • Delete: comment author or admin can delete a comment
  • Comments filter: show/hide all comment badges with one toggle
Comments Survive Version Restore
If you restore an older canvas version, comments on objects that exist in that version are preserved. Comments on deleted objects are archived.
Comment Notifications
  • You're notified when: someone @mentions you in a comment, someone replies to your comment, a comment is added to an object you created
  • Notification: in-app notification + email digest
  • Email digest: all new whiteboard comments grouped into one daily email (configurable)
  • In-meeting: if whiteboard is open in a meeting and someone comments, a small toast shows in the meeting UI
  • Comment activity also visible in Hub → Notifications timeline
