Rich Text Editor

A block-based rich text editor with full markdown support, slash commands for instant formatting, and a toolbar for quick access to headings, lists, tables, code blocks, and more.

Rich Text Editor
Text Formatting
  • Headings: H1 / H2 / H3 / H4
  • Bold, Italic, Underline, Strikethrough, Code inline
  • Text color and highlight color
  • Hyperlinks: Cmd+K → paste URL
  • Bullet list / Numbered list / Checklist
  • Blockquote / Horizontal rule
  • Font size: Small / Normal / Large
Block Types
  • Code block: with syntax highlighting (50+ languages)
  • Table: rows × columns with resize and merge
  • Image: paste or insert from Drive
  • Callout: colored info/warning/error box
  • Toggle / Accordion: collapsible section
  • Math: LaTeX equation support
  • Embed: any URL as an inline card
Slash Commands
/ Command Reference
/h1 /h2 /h3Heading levels 1–3
/bullet /numbered /checklistList types
/table [rows] [cols]Insert a table
/code [language]Code block with syntax highlight
/quoteBlockquote block
/callout [info/warn/error]Colored callout box
/imageOpen file picker or paste URL
/embedPaste URL for rich card embed
/ai [prompt]AI writes content at cursor (Writing Mode)
/tocInsert auto-generated table of contents
/dividerHorizontal rule separator
