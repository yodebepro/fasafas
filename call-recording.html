Message Translation
  1. Hover any message
    A language icon (🌐) appears in the message action bar alongside emoji, reply, and ⋯ more
  2. Click 🌐 Translate
    Message text is instantly replaced with the translated version
  3. 'Show original' link below the translation
    Click to toggle back to the original language
  4. Your preference remembered
    Messages from that sender automatically shown in translated form (optional setting)
Powered by DeepL + Google Translate
FasaFas uses DeepL for highest quality on European language pairs and Google Translate as fallback for broader language coverage. Translation happens server-side — original messages stored as-is.
Language Settings
🌐 Configuration
Default languageSet in Profile → Preferences → Language. All translations target this language.
Override per messageClick Translate and select a different target language from the dropdown
Supported languages50+ via DeepL, 120+ via Google Translate fallback
Source language detectionAutomatic — no need to specify the original language
Translation qualityDeepL for European pairs, Google Translate for Asian/African languages
Auto-Translate Setting
  • Enable: Settings → Language → Auto-translate messages not in [your language]
  • When on: messages detected as non-English (or non-your-language) are automatically translated
  • Original text always preserved and accessible via 'Show original'
  • Auto-translate only activates in channels where you've enabled it — not account-wide by default
  • Admin can set auto-translate as default for all new users
