Custom Emoji

Upload custom emoji — static images or animated GIFs — that work across all channels in your account. Name them with aliases, reuse them as reactions, and control uploads via admin approval.

Adding Custom Emoji
  1. Go to Settings → Emoji → Add Custom Emoji
  2. Upload a PNG, JPG, or GIF file (max 128×128px, 64 KB recommended)
    Larger images auto-resized. GIFs animate in chat.
  3. Give it an alias, e.g. ':fasa-logo:' or ':party-parrot:'
    Alias must be unique within your account. No spaces — use hyphens.
  4. Submit. If admin approval required, pending until approved.
  5. Once approved, visible to all workspace members immediately.
Using Custom Emoji
  • Type the alias: :party-parrot: renders the emoji inline
  • Emoji picker: click the smiley face → Custom tab shows all account emoji
  • Use as reactions: hover any message → + → search your custom emoji
  • Emoji auto-complete: type : and start typing the alias
  • Custom emoji work in: channel messages, DMs, threads, status field, Doc text
Admin Controls
  • Admin Panel → Emoji → review pending submissions
  • Approve or reject custom emoji before they go live
  • Set policy: open (anyone can add) or approval required
  • Delete existing custom emoji
  • Export all custom emoji as a ZIP
  • Set a maximum emoji count per user (prevents abuse)
  • Custom emoji stored in your account's S3 — not shared with other workspaces
