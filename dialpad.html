Call Queues

Set up call queues so inbound callers wait in line while agents handle calls — with custom hold music, estimated wait time announcements, overflow routing, and real-time queue monitoring.

Setting Up a Call Queue
  1. Admin Panel → Phone → Call Queues → Create Queue
  2. Name the queue and assign members
    e.g., 'Sales Queue' → add 5 sales reps as members
  3. Configure hold music
    Upload custom audio or use default FasaFas music
  4. Set maximum queue length
    Callers beyond this limit → routed to overflow destination
  5. Set maximum wait time
    Callers waiting longer → routed to overflow (voicemail or alternate number)
  6. Enable EWT announcement
    'Your estimated wait time is 4 minutes' — announced every 60 seconds
Routing Strategies
📋 Queue Routing Types
Sequential (Fixed Order)Rings Agent 1 first, then Agent 2, then Agent 3 — until someone picks up. Consistent priority.
Round-RobinDistributes calls evenly across all agents in rotation — fair distribution.
Least-IdleRoutes to the agent who has been idle longest since their last call — most balanced.
Skills-BasedRoute to agents with specific skills/tags (e.g., 'Spanish speaker', 'Enterprise' — Enterprise plan).
Priority QueueVIP callers (identified by CLI/number) jump to the front of the queue.
Real-Time Queue Monitoring
  • Supervisor dashboard: live view of all queues, waiting callers, agent status
  • Metrics shown: current queue length, average wait time, longest wait, abandonment rate
  • Agent status per queue: Available, In-Call, Wrap-Up, Away
  • Barge in: supervisor can join an active call silently (whisper coaching to agent)
  • Historical reports: daily/weekly/monthly queue stats — export CSV
  • Alerts: notification when queue exceeds 5 callers or average wait exceeds 3 minutes
