How PSTN Calling Works
  • FasaFas connects to the PSTN (Public Switched Telephone Network) via SIP trunking with certified carrier partners
  • Inbound calls: your business number → FasaFas SIP gateway → routed to the right user or IVR
  • Outbound calls: user places call in app → FasaFas SIP gateway → PSTN carrier → dialed number
  • Audio codec: G.711 (standard PSTN quality) or Opus (if carrier supports it)
  • Call path is encrypted with TLS + SRTP between app and gateway (last mile to PSTN is carrier-dependent)
  • Latency target: <150ms round-trip for excellent voice quality

Supported Devices
Soft Clients (FasaFas App)
  • Windows & Mac desktop app — calls via computer mic/speaker
  • Browser-based: Chrome/Edge/Firefox — WebRTC audio
  • iOS 15+ and Android 11+ mobile apps — push notification for incoming calls
  • Simultaneous ring: all your devices ring for incoming calls
Physical IP Phones (SIP)
  • Compatible with Polycom, Cisco, Yealink, Grandstream
  • Provision via admin: Admin Panel → Phones → Add Device → auto-provision URL
  • Calls routed identically to soft clients
  • Use alongside the app — all device types ring simultaneously
