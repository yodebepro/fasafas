Custom Intake Form Questions

Add custom questions to your booking form — so guests provide the information you need before the meeting, saving setup time and letting you prepare effectively.

Custom QuestionsPre-Meeting InfoCRM SyncConditional LogicRequired/Optional

Question Types
Using Responses
Intake Form Builder
  • Booking Page settings → Intake Form → Add Questions
  • Drag to reorder questions
  • Mark questions as Required (must answer to book) or Optional (can skip)
  • Preview: see exactly what guests will see before publishing
  • Saved responses appear in the event description and in the confirmation email you receive
  • All responses searchable in Hub → Calendar → Bookings
Available Question Types
📋 Question Types
Short textOne-line answer — for name, company, phone number
Long textMulti-line answer — for describing their use case or question
Single selectDropdown or radio buttons — 'What are you booking for: Demo / Support / Partnership'
Multi selectCheckboxes — 'Which products are you interested in: A / B / C'
Phone numberValidated phone field with country picker
Website URLValidated URL field — for their company website
Conditional questionShow a question only if a previous answer matches a condition
Using Form Responses
  • Responses included in your confirmation email: 'Maria booked a Demo. Company: Acme Corp. Use case: API integration.'
  • Responses appear in the event description in your calendar
  • Push to CRM: if Salesforce/HubSpot is connected → responses create or update a contact record automatically
  • Use in AI preparation: before the meeting, AI uses the intake form responses to generate a briefing card for you
  • Booking webhook: POST all responses to your endpoint when a booking is confirmed — for custom CRM or workflow integration
