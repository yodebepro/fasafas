Day / Week / Month / Agenda Views

View your schedule in four layout modes — Day for detailed hourly planning, Week for the full picture, Month for an overview, and Agenda for a clean chronological list — with color-coded calendars overlaid.

Calendar View Modes
📅 Layout Options
Day ViewHourly timeline for a single day. All events with time, attendees, and quick join button.
Week View7-day grid (Mon–Sun). Drag events to reschedule. Multi-day events span across days.
Month ViewFull month overview. Event titles shown; click for detail. Busy days visually darker.
Agenda ViewChronological list of upcoming events — no grid. Best on mobile. Infinite scroll forward.
Multi-person ViewAdd colleague calendars as overlays — see their free/busy alongside yours (with their permission).
Calendar Layers
  • Your FasaFas calendar: events created in FasaFas or synced from Google/Outlook
  • Google Calendar layer: shows synced events in their original colors
  • Microsoft Outlook/Exchange layer: shows work events synced from Outlook
  • Other people's calendars: overlay a colleague's free/busy (availability only, not event titles, unless shared)
  • Color coding: each calendar layer gets a distinct color — visible in all views
  • Toggle layers: show/hide any calendar layer with one click in the left sidebar
Navigation
  • Arrow buttons: navigate forward/backward by day, week, or month
  • Today button: jump to the current date instantly
  • Mini calendar (top left): click any date to jump — month view for quick navigation
  • Keyboard shortcuts: J/K for next/previous period, T for today, D/W/M/A for view modes
  • Search: Cmd+F / Ctrl+F to search event titles and descriptions
  • URL: /calendar/2026-05-01 — shareable link to a specific date
