RSVP Tracking

Track who's attending your meeting in real time — see who accepted, declined, or hasn't responded, send reminders to non-responders, and get an attendance report after the event.

Attendee RSVP Experience
  • Invitee receives an email: Accept / Decline / Maybe buttons — one click, no sign-in required
  • Or: accepts via calendar app (Google Calendar, Outlook) — response synced back to FasaFas
  • In-app notification: FasaFas users see an 'Invitation pending' badge on the event
  • Response options: ✅ Accept · ❌ Decline · ❓ Maybe · Add a note with your response
  • Change response: invitee can update RSVP at any time before the meeting
  • Add to calendar: all email invites include .ics attachment for non-FasaFas calendar users
Host RSVP Dashboard
  • Event detail → Attendees tab shows real-time RSVP status per person
  • Status badges: ✅ Accepted · ❌ Declined · ❓ Maybe · ⏳ No response
  • Filter: show only non-responders to send a targeted reminder
  • Send reminder: select non-responders → 'Send Reminder' → custom email sent
  • Re-invite: add a new invitee to an existing event → they receive invitation email immediately
  • Remove attendee: remove from event without canceling for everyone else
Post-Event Attendance Report
  • After the meeting ends, FasaFas generates an attendance report automatically
  • Report includes: who actually joined the meeting vs who RSVP'd yes
  • Join time, leave time, and total duration per participant
  • Report emailed to the event organizer + available in Hub → Calendar → Reports
  • Compare: invited vs accepted vs attended — useful for recurring team meetings
  • Export: CSV download for HR or compliance reporting
