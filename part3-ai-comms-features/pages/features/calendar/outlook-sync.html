Outlook / Exchange Sync

Connect Microsoft Outlook or Exchange Server — personal and room calendars sync bidirectionally, RSVPs propagate, and you can book conference rooms directly from FasaFas.

Outlook / Exchange Sync Features
  • Same bidirectional sync as Google — events, RSVPs, and deletions sync in both directions
  • Supports: Office 365 (cloud) and Exchange Server (on-premise, 2016+)
  • Sync scope: up to 90 days history on initial connect; real-time going forward
  • Meeting categories / labels: synced with Outlook color categories
  • Recurrence patterns: weekly, monthly, custom patterns all sync correctly
  • Private events: appear as 'Busy' in FasaFas without sharing title or description
Setup
  1. Settings → Integrations → Microsoft Calendar → Connect
  2. Sign in with Microsoft (Office 365) or enter Exchange Server URL + credentials
  3. Grant calendar access permissions
  4. Select calendars to sync (personal + any shared/room calendars you manage)
  5. Initial sync completes in 1–3 minutes
🔧 Exchange Server Support
VersionsExchange 2016, 2019, and Exchange Online (Office 365)
ProtocolEWS (Exchange Web Services) — standard Microsoft protocol
AuthenticationModern Auth (OAuth2) preferred; NTLM supported for legacy Exchange
On-premiseRequires FasaFas connector agent installed on your network (Enterprise plan)
Room & Resource Booking
  • If you have access to room calendars in Exchange, they sync and appear in FasaFas
  • When creating an event in FasaFas → 'Add Rooms' → browse your Exchange room directory
  • Select available rooms (real-time availability shown)
  • Room is booked in Exchange automatically when you save the FasaFas event
  • Room conflicts: FasaFas shows rooms as unavailable if already booked in Exchange
  • Cancel event in FasaFas → room booking automatically cancelled in Exchange
