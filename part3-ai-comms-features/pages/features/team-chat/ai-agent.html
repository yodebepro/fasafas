Custom Agents in Channels
  • Any agent you build in Agent Builder can be invoked via @mention in channels
  • Trigger an agent: @FasaFas run [agent-name]
  • Or configure the agent to auto-post results to a specific channel when triggered
  • Example: meeting-ended agent → posts summary to #product channel automatically
  • Agents can be given a custom name and avatar so they appear as a separate bot (Enterprise)
  • Admin manages which agents are available in which channels
