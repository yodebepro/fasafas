Real-Time Translation Pipeline
  1. Whisper transcribes the original language
    1–2 second delay from speech to original-language text — same as standard captions
  2. Translation API called per caption segment
    Each segment sent to DeepL (European languages) or Google Translate (Asian/African languages). Smart routing selects best provider per pair.
  3. Translated text returned in ~200–400ms
    Total display latency: 1.5–2.5 seconds. Translation adds ~500ms on top of standard caption delay.
  4. Per-subscriber delivery
    Each participant sees their chosen language. Participant A sees French, B sees Spanish — from the same English presenter. Independent per viewer.
