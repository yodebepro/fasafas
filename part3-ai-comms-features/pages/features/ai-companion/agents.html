Agent Builder

Build custom AI agents with a no-code visual flow editor — connect triggers (meeting ended, file uploaded) to AI actions (send email, create task, post to Slack) in minutes, with no programming required.

Agent Builder Interface
  • Canvas-based drag-and-drop flow editor — zero code required
  • Each flow: one Trigger → optional Conditions → optional AI Steps → one or more Actions
  • Add Conditions for branching: 'if meeting had 5+ participants, then…'
  • Add AI Steps between trigger and action: 'summarize transcript before sending email'
  • Test with a sample event before activating
  • View run history: succeeded, failed, skipped — with full payload logs
  • Agents run on FasaFas infrastructure — no external server or cron job needed
Available Triggers & Actions
Triggers (When)
  • 🎥 Meeting ended
  • 📝 Summary ready
  • ☁️ Recording completed
  • 📁 File uploaded to channel
  • 📅 Daily/weekly schedule (cron)
  • ✅ Action item overdue
  • 💬 Keyword mentioned in chat
  • 📞 Voicemail received
  • 🔗 External webhook received
Actions (Then)
  • 📧 Send email with AI-generated content
  • 💬 Post to Slack channel
  • ✅ Create task in Tasks / Asana / Jira
  • 📄 Create or update a Doc
  • 📁 Move file to Google Drive / OneDrive
  • 🤖 Run a custom AI prompt
  • 🔗 Call a webhook (HTTP POST)
  • 📊 Update a spreadsheet row
  • 🔔 Send in-app notification
Common Agent Examples
Auto Follow-Up Email
  • Trigger: Meeting ended
  • AI Step: Generate follow-up email from summary
  • Action: Send email to all attendees
Slack Summary Bot
  • Trigger: Summary ready
  • Action: Post summary excerpt to Slack channel
  • Includes: decisions, actions, recording link
Overdue Task Reminder
  • Trigger: Action item overdue (daily)
  • Action: Notify assignee
  • Escalation: Also notify manager after 3 days
