Share Files

Push files, documents, whiteboards, or links to all breakout rooms simultaneously — or to a single room. Every participant receives the material instantly without leaving their room.

What Can Be Shared to Rooms
Supported Content Types
  • FasaFas Docs — share a collaborative doc (participants can open and edit immediately)
  • Whiteboards — share a whiteboard link (participants can collaborate in real time)
  • External Links — any URL (opens in participants' browser)
  • Files — PDF, DOCX, PPTX, images (download link appears in room chat)
  • Clips — async video clips (plays inline in room)
  • Poll — launch a poll in all rooms simultaneously
File Size Limits
  • PDF / DOCX / PPTX: up to 512 MB per file
  • Images (JPG, PNG, GIF): up to 100 MB
  • Videos (MP4): up to 1 GB
  • Links: no size limit — just a URL
  • Total per session: no limit (storage quota applies separately)
How to Share Files to Rooms
  1. Open Breakout Rooms panel
    From the main session. Files can only be pushed from the main session — not from inside a breakout room.
  2. Click "Share to All Rooms" or "Share to Room X"
    The Share panel opens. Choose whether to send to all rooms at once or select individual rooms.
  3. Select content
    Browse your FasaFas files (Docs, Whiteboards, Clips) or upload a new file. Paste a URL for external links.
  4. Click Share
    The file or link appears instantly in every selected room's chat as a highlighted message from "Host". Participants receive a notification chime.
Sharing to a Single Room
You can also join a specific room first, then share your screen or upload a file from inside that room — this only affects participants in that room. Use this for room-specific materials that shouldn't go to everyone.
How Participants Receive Shared Files
  • A notification banner appears at the top of their room: "The host has shared a file"
  • The file link or Docs/Whiteboard link appears as a pinned message in their room chat
  • Click the link to open the file — it opens in a new browser tab or inline (for Docs/Whiteboards)
  • For collaborative Docs and Whiteboards: all participants in the room can edit simultaneously
  • Shared files remain in room chat history for the duration of the session
  • Participants can re-access files by scrolling up in the room chat at any time
  • Files are not deleted when the room closes — they remain accessible in participants' Hub after the meeting
