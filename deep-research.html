Mind Map Generation

Generate an interactive mind map from any document's content with one click — AI identifies the main topics and sub-points and renders them as an explorable visual in a side panel.

Generate Mind Map from Doc
  1. Open any doc → Doc menu (⋯) → Generate Mind Map
  2. Or: type /mindmap in the doc body
  3. AI reads the full document → identifies main headings as branches → sub-headings and key concepts as children
  4. Mind map renders in a side panel alongside the doc — doesn't replace the doc view
  5. Generation time: 5–15 seconds depending on doc length
  6. Re-generate: click 'Refresh' if doc content has changed since last generation
Interacting with the Map
  • Pan and zoom the mind map like a whiteboard
  • Click any node → doc scrolls to that section (bidirectional link)
  • Expand/collapse branches: click the circle on any node
  • Edit labels: double-click any node → type to rename
  • Add new nodes: Tab (child) / Enter (sibling) — same as Whiteboard mindmap
  • Delete nodes: Delete key — removes from mind map but not from doc
  • Changes to the mind map don't update the doc — it's a visualization, not the source of truth
Export & Sync to Whiteboard
  • 'Open in Whiteboard' button → mind map transferred to a new FasaFas Whiteboard
  • In whiteboard form: full collaboration, cursors, CRDT sync, and all whiteboard tools available
  • Export from panel: PNG / SVG / OPML
  • Share mind map link: independent shareable URL for the visualization (view-only)
  • Embed in another doc: /embed → paste mind map URL → renders interactive mind map inline
