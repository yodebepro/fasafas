Gallery View

Gallery View shows all participants as equal-size video tiles in a grid — up to 49 tiles visible per page. Paginate through pages to see everyone in large meetings.

Gallery View Layout
Grid Specifications
  • Up to 49 participant tiles visible per page (7×7 grid)
  • Tile size scales based on screen resolution and number of participants
  • 1–4 participants: large tiles with plenty of space
  • 5–9 participants: 3×3 grid arrangement
  • 10–25 participants: 5×5 grid
  • 26–49 participants: 7×7 grid with pagination if more
  • Pagination arrows appear at edges when >49 participants
What Each Tile Shows
  • Participant's video stream (480p in gallery, upgrades to 1080p when pinned)
  • Participant name (bottom-left overlay)
  • Mute indicator (microphone icon, red when muted)
  • Hand raise indicator (✋ icon)
  • Connection quality indicator (signal bars)
  • Initials avatar shown when camera is off
Gallery View Controls
  • Switch to Gallery: click View → Gallery (or keyboard shortcut Alt+G on Windows)
  • Hide participants who have video off: View → "Hide non-video participants"
  • Pin a tile: hover a tile → click Pin — that tile stays in corner when speaker changes
  • Spotlight (host only): right-click tile → Spotlight for All — forces everyone to see this person
  • Multi-pin: pin up to 9 tiles simultaneously by pinning multiple participants
  • Scroll pages: keyboard arrow keys or click pagination arrows
  • Click any tile name to open participant options (mute, rename, remove)
Performance at Scale
Lazy loading for large meetings
In meetings with 100+ participants, FasaFas only downloads video streams for the tiles currently visible on screen. Tiles on other pages are dormant until you paginate to them — significantly reducing download bandwidth and CPU usage.
📊 Bandwidth per Gallery Page
9 visible tiles~4 Mbps download (9 × 480p streams at ~440 kbps each)
25 visible tiles~8 Mbps download (25 × 240p streams at ~320 kbps each)
49 visible tiles~11 Mbps download (49 × 240p at ~225 kbps, adaptive per network)
