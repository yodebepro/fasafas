Storage pooled across team
On Pro and Business plans, storage is pooled. If your account has 20 Pro licenses → 200 GB total shared pool. One user can use 150 GB and another 50 GB — it's the total that matters, not per-user.
Managing Your Storage
  • View usage: Admin Panel → Billing → Storage → shows used vs total with per-user breakdown
  • Largest recordings: Admin Panel → Recordings → sort by file size to find space hogs
  • Alerts: automatic notification emails at 80% and 95% usage
  • Per-user quota: admin can cap individual users (e.g., Sales team max 5 GB each)
  • Delete recordings: Hub → Recordings → select → Delete (moves to Trash, purged after 30 days)
  • Restore from Trash: Hub → Recordings → Trash → Restore (within 30-day window)
  • Archive to cold storage: recordings older than X months automatically moved to Glacier (lower cost, slower retrieval)
