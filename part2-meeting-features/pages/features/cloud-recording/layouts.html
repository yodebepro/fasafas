Recording Layout Options
🎬 Available Recording Layouts
Active SpeakerCurrent speaker shown in a large main frame. Switches automatically as speakers change. Best for most meetings. Default.
Gallery ViewGrid of all participants on camera. Fixed — doesn't change when someone speaks. Captures group dynamics.
Shared Screen OnlyRecords only the content being shared (slides, app, browser). No participant video. Smallest file size. Best for software demos.
Speaker + Screen (Side-by-Side)Split view: shared screen on left, active speaker thumbnail on right. Great for presentations with a visible presenter.
Audio Only (M4A)No video — just the audio track. Very small file. Good for podcast-format or voice-memo use cases.
How to Select Your Recording Layout
  • During a meeting: click Record → Record to Cloud → select layout from dropdown before clicking Start
  • In settings: Settings → Recording → Cloud Recording → check which layouts to enable as options
  • Per-meeting default: in the Schedule Meeting form, select the default layout for that specific meeting
  • Multiple layouts simultaneously: check all desired layouts — FasaFas records all of them in parallel (each produces a separate file)
  • Admin can restrict which layouts are available to prevent non-compliant recordings
  • Layout selection cannot be changed after recording has started — you must stop and restart with the new layout
