Action Item Extraction

FasaFas automatically detects action items, tasks, and commitments from the meeting transcript — then structures them with suggested assignees and due dates, ready to push to Tasks or any project management tool.

How Action Items Are Detected
  • Claude scans the full transcript for commitment and assignment language patterns:
  • Commitment phrases: "I'll take care of…", "I will send…", "I'm going to…", "I'll follow up on…"
  • Assignment phrases: "Can you [name]…", "[Name] will handle…", "Action for [name]:"
  • Deadline phrases: "by Friday", "end of day", "next sprint", "before the 15th", "this week"
  • Decision-action pairs: "We decided to [X] — [Name] will lead it"
  • Each detection includes: task description, speaker (suggested assignee), timestamp in recording
  • Confidence score attached: High / Medium / Needs Review
  • "Needs Review" items appear in a separate section requiring host confirmation

Pushing Action Items to Tasks & Integrations
FasaFas Tasks
  • One-click: "Push All Action Items to Tasks" from the summary page
  • Or individually push each item with the → Tasks button
  • Assignee, due date, and source meeting pre-filled
  • Assignee receives a notification with the task and meeting link
  • Task completion tracked in the AI daily digest
External Tool Integration
  • Asana: creates task in specified project, assignee mapped by email
  • Jira: creates issue in specified board, description includes transcript quote
  • Monday.com: creates item in specified board
  • Notion: appends to a specified database page
  • Zapier: webhook trigger on each action item → any destination
