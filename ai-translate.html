Threaded Replies

Reply to any message in a thread to keep channel conversations organized — threads collapse in the main view and can be followed for notifications without cluttering the channel.

Best Practices
How Threaded Replies Work
  • Hover any message → click 'Reply in thread' (or press T)
  • Thread opens in a panel on the right side of the screen
  • Main channel shows: original message + reply count + last reply time
  • Thread replies never clutter the main channel flow
  • 'Also send to channel' checkbox: optionally show a reply in both thread and channel
  • Thread participants automatically follow the thread and receive notifications
  • Jump to thread: click the reply count below any parent message
Thread Notifications
🔔 Thread Notification Modes
FollowingYou receive notifications for all new replies (auto if you sent a message or reacted)
Not followingNo notifications — you can still read the thread anytime
@mentionedAlways notified if someone @mentions you in a thread, regardless of following status
UnfollowClick 'Following' toggle in thread header to unfollow
Thread Best Practices
  • Use threads for: feedback on a specific proposal, back-and-forth Q&A, progress updates on a task
  • Keep the channel for: announcements that everyone needs to see immediately
  • Use 'Also send to channel' sparingly — for final decisions that the whole channel should know
  • Threads older than 7 days with no new activity collapse by default in the sidebar
  • AI can summarize an entire thread: hover thread header → 'Summarize with AI'
