Timer & Auto-Close

Set a timer so breakout rooms automatically close after a defined period. A visible countdown appears for all participants, and a warning banner appears before the time is up.

Setting Up the Timer
Timer Configuration
  • Enable timer: toggle Set a timer for breakout rooms before opening rooms
  • Duration: type any value from 1 to 120 minutes
  • Timer starts as soon as you click Open All Rooms
  • Timer applies to all rooms simultaneously — not per-room
  • All rooms close at the same time when timer hits 0
  • Host can close rooms early at any point regardless of timer
Warning Settings
  • Default warning: 60 seconds before auto-close
  • Configurable: 30 seconds / 1 minute / 2 minutes
  • Warning appears as a red banner at top of participant's screen
  • Warning banner text: "Breakout rooms will close in X seconds"
  • Audio chime plays with the warning (configurable)
  • Host receives a separate alert that time is almost up

What Participants See
During the Session
  • A small countdown timer appears in the top-right corner of their room view
  • Format: "14:32 remaining" updating every second
  • Timer is always visible — participants can manage their time accordingly
  • If host extends time, timer updates live with the new remaining time
At Warning Time
  • A bold red/amber banner appears across the top of their screen
  • "⚠️ Breakout rooms will close in 60 seconds. Please wrap up."
  • Audio chime plays once (can be muted in personal settings)
  • Participants should save any notes, share conclusions in chat, etc.
  • No action required from participants — they're automatically returned
What happens at exactly 0:00?
All breakout rooms close simultaneously. Every participant is automatically moved back to the main session — no manual action needed by host or participants. Video and audio reconnect to the main session immediately.
Extending Time Mid-Session
  1. Open Breakout Rooms panel
    The panel shows the current timer countdown alongside each room's participant count.
  2. Click "Add Time"
    A small dialog lets you add time in 1-minute increments or type a custom value.
  3. Confirm
    The timer updates immediately for all participants in all rooms. Their countdown resets to the new extended time. All participants see the timer update live.
Participants Can Request an Extension
Participants can click Ask for More Time in their room (if the host enables this). The host receives a notification and can choose to extend with one click from the main session.
