Semantic Search

Search finds what you mean, not just what you typed — semantic search understands intent, finds conceptually related messages even without matching keywords, and searches across channels, DMs, files, and meeting transcripts.

Semantic vs Keyword Search
  • Keyword: 'budget approval' only finds messages containing those exact words
  • Semantic: 'budget approval' also finds 'finance sign-off', 'Q3 spend confirmed', 'cost approved by leadership'
  • Technology: text-embedding-3-large encodes every message into a 3,072-dimensional vector. Stored in pgvector.
  • At search time: your query is embedded and compared to all message vectors via cosine similarity
  • Results ranked by semantic relevance + recency + your engagement with the source channel

What Gets Searched
  • Channel messages and thread replies
  • Direct messages and group DMs (only your conversations)
  • File names and AI-extracted file content (PDFs, Docs)
  • Meeting transcript text from all meetings you attended
  • FasaFas Docs content
  • Voice message AI transcripts
  • Custom emoji names
  • @mentions of your name
Search Tips
  • Use natural language: 'find the message about delaying the launch' works better than 'launch delay'
  • Filter by: date range, channel, sender, file type — use the filters panel on the left
  • Operator: in:#channel limits search to a specific channel
  • Operator: from:@name limits to messages from a specific person
  • Operator: before:2026-01-01 or after:2026-01-01 for date filtering
  • Operator: has:file limits to messages with file attachments
  • Shortcut: Cmd+K (Mac) or Ctrl+K (Windows) opens universal search from anywhere
