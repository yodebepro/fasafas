Inline Comments & Suggestions

Leave inline comments on any text selection, use Suggestions Mode to propose edits that the author can accept or reject, and @mention teammates to pull them into a discussion.

Managing Feedback
Inline Comments
  • Select any text → click the comment icon (💬) that appears → type your comment
  • Comment anchored to that text selection — highlighted in yellow in the doc body
  • Reply in thread: others can reply directly in the comment thread
  • @mention: type @ → name autocomplete → they receive a notification
  • Resolve: click ✓ to mark a thread resolved — it collapses and hides from the main view
  • Comment panel: right sidebar lists all open comment threads — click to jump to each
  • Comments are page-persistent: they survive formatting changes and edits to surrounding text
Suggestions Mode (Track Changes)
  • Enable: View menu → Suggestions Mode (or Cmd+Shift+X)
  • In Suggestions Mode: every edit you make is tracked — not applied directly
  • Insertions: shown in green underline
  • Deletions: shown in red strikethrough
  • The document owner sees all suggested changes highlighted
  • Accept: click ✓ next to any suggestion → change applied to the document
  • Reject: click ✗ → change discarded, original text restored
  • Accept all: 'Accept all suggestions' button — applies every tracked change at once
Managing Comments & Suggestions
  • Comment panel (right sidebar): see all threads — filter open / resolved / mine / @mentions
  • Jump to any comment: click in panel → doc scrolls to that section
  • Resolve all: admin/owner can resolve all open comments at once
  • Export feedback: download all comments as a CSV or push to a FasaFas Doc summary
  • Notifications: you're notified when someone replies to your comment or @mentions you
  • Email digest: daily digest of all unread doc comments across your workspace
